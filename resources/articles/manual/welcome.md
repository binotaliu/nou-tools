# 歡迎使用 NOU 小幫手

NOU 小幫手是一款專為 NOU 學生設計的非官方工具，旨在幫助你更好地管理課表與學習進度。

## 主要功能

### 課表管理

你可以透過 NOU 小幫手建立屬於自己的課表：

- **快速檢視**：一目了然地看到所有視訊面授時間
- **WebCal 訂閱**：將課表同步到你的日曆應用程式
- **多種檢視方式**：以列表或日曆的方式檢視課表

### 學習進度追蹤

記錄每週的學習進度，包含：

- 課本閱讀進度
- 影音課程觀看進度
- 個人備註與心得

## 開始使用

1. 點擊「建立我的課表」按鈕
2. 選擇你要修讀的課程
3. 系統會自動為你整理課表資訊

## 需要協助？

如果在使用過程中遇到任何問題，歡迎查閱我們的操作手冊，或是透過 GitHub Issue 與我們聯繫。
