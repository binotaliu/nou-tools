# Security Policy

This repository follows responsible disclosure practices. If you discover a security
vulnerability, please report it so we can resolve it promptly and keep users safe.

## Reporting a Vulnerability

- **Email:** Send details to [me@binota.org](mailto:me@binota.org)
- **PGP:** Use the public key available at `https://binota.org/pgp-key.txt` to encrypt
  sensitive information.

Include as much information as possible:

1. Affected versions or configurations.
2. Steps to reproduce the issue.
3. Potential impact and any mitigations you've already tried.

We will acknowledge receipt of your report and respond as quickly as possible. Please
allow us time to investigate and coordinate a fix before disclosing details publicly.

## Scope

This policy applies to the code in this repository and any associated web services.
Third-party libraries may have their own security processes; please follow their
recommendations as well and notify us if you believe they affect this project.

## Acknowledgements

Security issues that are reported and fixed may be publicly acknowledged with
your consent. See [public acknowledgements](https://github.com/binotaliu/nou-tools/blob/main/SECURITY.md).

## Disclaimer

We are not responsible for any damage caused by vulnerabilities in software you
create using this project. Always follow best practices and keep dependencies up
to date.
